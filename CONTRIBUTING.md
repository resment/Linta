# Contributing

By contributing to Linta, you agree that your contribution may be distributed under:

1. the PolyForm Noncommercial License 1.0.0; and
2. separate commercial licenses offered by the project owner.

This dual-license contribution grant is required so the project can remain free for noncommercial
use while preserving the project owner's right to offer paid commercial licenses.

## Contribution Rules

- Do not submit code you do not have the right to contribute.
- Do not add third-party code with incompatible license terms.
- Keep tests deterministic and do not call external LLM APIs.
- Keep `README.md` and `README_CN.md` synchronized when behavior or licensing changes.
- Include tests for new CLI behavior.

If you do not agree to the dual-license contribution grant, do not submit a contribution.
